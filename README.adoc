= Asciidoc Renderer for Hexo

Add support for Asciidoc. This plugin uses link:https://www.npmjs.com/package/asciidoctor.js[asciidoctor.js] as render engine.

== Install

----
$ npm install hexo-renderer-asciidoc --save
----
